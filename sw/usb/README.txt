Readme: usb
-----------
- This directory contains the files to work with usb to read images and 
  transfer them through USB
- To use this you must install libusb (on OSX, simply run "brew install libusb"
- The Makefile has been tested on OSX 10.10, you will have to mess around
  with the compile link/flags on another machine

Software Details
----------------
- libusb 1.0.19
